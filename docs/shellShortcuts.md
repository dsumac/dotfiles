# Shell shortcuts

Keyboard shortcuts for command-line editing (readline / zsh).

## Cursor movement

| Shortcut | Action                              |
| -------- | ----------------------------------- |
| `Ctrl+a` | Move cursor to start of line        |
| `Ctrl+e` | Move cursor to end of line          |
| `Ctrl+b` | Move back one character             |
| `Ctrl+f` | Move forward one character          |
| `Alt+b`  | Move back one word                  |
| `Alt+f`  | Move forward one word               |
| `Ctrl+xx`| Toggle between first and current position |

## Editing

| Shortcut  | Action                              |
| --------- | ----------------------------------- |
| `Ctrl+d`  | Delete current character            |
| `Ctrl+w`  | Cut the last word                   |
| `Ctrl+k`  | Cut everything after the cursor     |
| `Ctrl+u`  | Cut everything before the cursor    |
| `Alt+d`   | Cut word after the cursor           |
| `Alt+w`   | Cut word before the cursor          |
| `Ctrl+y`  | Paste the last cut text             |
| `Ctrl+_`  | Undo                                |

## History

| Shortcut  | Action                                         |
| --------- | ---------------------------------------------- |
| `Ctrl+r`  | Search command history (type to filter)        |
| `Ctrl+j`  | End search, keep current history entry         |
| `Ctrl+g`  | Cancel search, restore original line           |
| `Ctrl+n`  | Next command in history                        |
| `Ctrl+p`  | Previous command in history                    |

## Terminal

| Shortcut  | Action                |
| --------- | --------------------- |
| `Ctrl+l`  | Clear the terminal    |
| `Ctrl+c`  | Cancel current command|
